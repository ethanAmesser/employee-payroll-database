# HR & Payroll Relational Database System

## Overview
Designed and implemented a relational HR and payroll database using Microsoft Access. The system manages employee records, calculates bi-weekly payroll, and generates HR reports through queries, forms, and macros.

## Technologies Used
- Microsoft Access
- SQL Queries
- Relational Database Design
- Macros
- Forms & Reports

## Key Features
- Employee and manager record management
- Automated bi-weekly payroll calculations
- Relational database structure with primary and foreign keys
- SQL queries for payroll processing and reporting
- Interactive forms and reports for HR operations

## Skills Demonstrated
- Relational database design
- SQL query development
- Data normalization
- Business data reporting
- Database automation with macros
